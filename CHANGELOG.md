# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.1.0] - 2026-02-26

### Added

- Button component with primary, secondary, and destructive variants
- Text component with typography variants and contrast levels
- TextInput with animated floating label and error handling
- SwitchToggle component
- SelectInput with bottom sheet picker
- DateTimeInput component
- SegmentedControl component
- Search input component
- Sheet and BottomSheet modal components
- TitleBar navigation component
- NavigationListRow component
- RadioOptionInput and RadioOptionItem components
- CheckboxOptionInput and CheckboxOptionItem components
- Chevron and DragHandle utility components
- MenuItemDescription component
- FieldError component
- Form components (Form, FormTextInput, FormSwitchToggle, FormSelectInput, FormSegmentedControl, FormDateTimeInput) with `informed` integration
- ThemeContext and useTheme/useThemedStyle hooks
- ATLAS_LIGHT default theme
- Animated border and font-size interpolation hooks
